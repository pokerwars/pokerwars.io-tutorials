To let you play against poker bots from all over the world, we created pokerwars.

pokerwars is none the less a platform where all poker bots can meet and play against each other. But how can they play? How can a bot fold? Or raise?

REST

Computers communicate over the network, so your poker bot will do. There are many ways to do this, but at pokerwars we chose to follow REST. REST is a set of principles that enforce how the machines communicate to each other. The REST operations used in pokerwars are:

  • GET: used to ask something to a machine
  • POST: used to say something to a machine

REST has some other operations too, have a read here if you want to learn more.

Java

In this tutorial you will use Java, a famous and very used programming language which follows the Object Oriented (OO) paradigm.

Java alone would be enough, but in order to get something done fast, we will also take advantage of an application framework. An application framework is a piece of software that helps you in solving some common programming problems faster. In our case we chose Spring Boot, which gives you a lot of power with very few lines of code. But remember, with power comes responsability! 😜
