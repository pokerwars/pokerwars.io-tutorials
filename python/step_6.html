Our first method

We are now going to add the first method with some basic logic. We start simple, with the ping endpoint. The desired behaviour is to receive a call from the pokerwars server, log it, and reply with a pong.

The syntax is the following: 

        def method_name(parameters):
        return "hello world"

In our case, inside the method we are going to log the call using the printl command with the following message: 

'Received ping from pokerwars.io, responding with a pong'

And finally the interesting part, replying back with a pong JSON content. This is how to do it 

        response.content_type = 'application/json'
        return {"pong": True}

Setting the content type to JSON and then invoking the return keyword with a the pong JSON response.
