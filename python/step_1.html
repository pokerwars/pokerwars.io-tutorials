Bottle

Bottle is a fast, simple and lightweight micro web-framework for Python. In this session we are going to show you how to define endpoints, receive data from a API call, manipulate data and (very important) how to make a call to an external API.

As you might already know from your previous python experience (or because you already checked out or starter bot) we need to import the bottle module along with the functions that we are going to use during this example. There are several interesting concepts in bottle:

  • Routing: Requests to function-call mapping with support for clean and dynamic URLs.
  • Templates: Fast and pythonic built-in template engine and support for mako, jinja2 and cheetah templates.
  • Utilities: Convenient access to form data, file uploads, cookies, headers and other HTTP-related metadata.
  • Server: Built-in HTTP development server and support for paste, fapws3, bjoern, gae, cherrypy or any other WSGI capable HTTP server.

If you want to know more about bottle you can find more information here.

Imports

We want to show you all the basics steps you need to know to become the new master of pokerwars, so we need to start right from the first step, you need to know your imports!!!

There are different way to import external libraries in python, in this case we decided to import just specific section of a module. To do so you have to combine the two keywords from and import in the following way. 

from pokerwars import winnerBot

in this way you are specifying to import just the section winnerBot from the module pokerwars, and not the rest of the module.

Now you know what to do, on your right you can see that we already add few other imports. What you need to do to complete your first step is to import the sections get, post, run, request from the bottle module. And remember, enjoy the ride!!!
